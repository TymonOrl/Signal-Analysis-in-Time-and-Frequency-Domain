# Signal Analysis in Time and Frequency Domain

A collection of Jupyter notebooks demonstrating techniques for analyzing signals in both the **time domain** and the **frequency domain**, including transforms and visualizations commonly used in digital signal processing (DSP).

## Overview

This project explores how signals can be represented and interpreted in different domains:

- **Time-domain analysis** — examining how a signal's amplitude changes over time (e.g. waveform plots, statistical measures).
- **Frequency-domain analysis** — examining how a signal's energy is distributed across frequencies, typically via the Fourier Transform (FFT).
- **Time-frequency analysis** — techniques (e.g. Short-Time Fourier Transform / spectrograms) that show how frequency content evolves over time.

## Topics Covered

- Signal generation and visualization
- Fast Fourier Transform (FFT)
- Power spectral density
- Windowing functions
- Time-frequency representations (e.g. spectrograms)
